# IOTA-MAM-Explorer

IOTA-MAM-Explorer example application for fetching IOTA-MAM-Data (Masked Autheticated Messaging)

## Getting Started

git clone https://github.com/ollibertinho/IOTA-MAM-Explorer && cd IOTA-MAM-Explorer && npm i

### Prerequisites

```
npm
nodejs
```
