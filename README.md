# Project Title

IOTA-MAM-Explorer example application for fetching IOTA-MAM-Data (Masked Autheticated Messaging)

## Getting Started

git clone git@github.com:ollibertinho/IOTA-MAM-Explorer.git && cd IOTA-MAM-Explorer && npm i

### Prerequisites

```
npm
nodejs
```
